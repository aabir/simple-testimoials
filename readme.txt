This is a simple testimonial plugin for WordPress. After instalation of the plugin a new menu will appear in WP admin, 
name Testimonial. To use it in a page use short code like [testimonials], you also can specify number of post to show in
the page, to display 4 post use the shortcode like this [testimonials per_page = "4"]. Simple testimonial plugin also
provide a slide widget to display testimonial in widget area.
