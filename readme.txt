This is a simple testimonial plugin for WordPress. After installation of the plugin a new menu will appear in WP admin, 
name Testimonial. To use it in a page use short code like [simple_testimonials], you also can specify number of post to show in
the page, to display 4 post use the shortcode like this [simple_testimonials per_page = "4"]. Simple testimonial plugin also
provide a slide widget to display testimonial in widget area.
